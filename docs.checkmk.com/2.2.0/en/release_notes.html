Update to version 2.2.0 Update matrix for version 2.2.0 Basic information on the installation of Checkmk Monitoring Linux Monitoring Windows

1. Introduction

This document lists the platforms and integrations supported by Checkmk 2.2.0. Except for very specific reasons (e.g. security), these platforms, including all patch versions will remain supported through active maintenance during the Checkmk 2.2.0 product lifecycle.

2. Server OS

Checkmk can be installed on the following Linux server OS distributions and versions running on x86_64 hardware. Entries might contain the following additions:

  • Deprecated: LTS/LTSS versions marked in this way are still supported for the entire life cycle of version 2.2.0. Support will be ceased afterwards. For Ubuntu STS versions, supplying Checkmk 2.2.0 packages ends shortly after Ubuntu stops supporting them with updates.

  • Once released: Support for these versions is planned. Checkmk 2.2.0 packages will be made available shortly after the release of the new distribution version.

We will additionally evaluate any new versions of these distributions as a part of our ongoing support process. These will be covered in the update matrix.

Important: Starting with Checkmk 2.3.0 Ubuntu STS versions will not be supported anymore.

3. Browser

The Checkmk user interface has been verified as able to be run on the following desktop browsers and minimum versions:

  • Google Chrome >= 100 on Windows, Linux and macOS

  • Firefox >= 100 on Windows, Linux and macOS

  • Microsoft Edge >= 100 on Windows

  • Safari >= 15.6.1 on macOS

The Checkmk mobile user interface has been verified to run on the following mobile browsers:

  • Safari on iOS (current and last major OS release)

  • Google Chrome on Android (current and last major OS release)

4. OS for the Checkmk agent

The Checkmk agent can be installed and run on the following operating systems if the OS meets some basic requirements.

4.1. Linux/Unix

All Linux and Unix operating system releases are supported, if at least the minimal version of these technical components as listed below is installed:

  • Bash >= 4.2

  • ksh >= 93

  • Perl >= 5.14.0

Additionally, one of the following components needs to be present:

  • systemd >= 219

  • xinetd >= all versions

Please be aware: Plug-ins may require a specific version of additional components to be functional. Python plug-ins require version 2.7 or > 3.5.

4.2. Windows

All 64-bit versions of the following Windows operating system releases are supported:

  • 2012

  • 2012 R2

  • 2016

  • 2019

  • 2022

For older releases, we provide a legacy agent with a limited feature set and without official support.

5. Grafana plug-in

Checkmk 2.2.0 supports the following Grafana plug-in versions:

  • >= 3.0.0

6. Checkmk Appliance

Checkmk 2.2.0 can run on the following versions of the Checkmk Appliance:

  • >= 1.6.3

7. LDAP server

With LDAP Checkmk 2.2.0 supports synchronization of the following LDAP servers:

  • Microsoft Active Directory

  • OpenLDAP

  • 389 Directory Server

8. SAML IdP

The SAML integration supports the following Identity Providers (IdPs):

  • Microsoft Azure Active Directory

  • Google Cloud

  • Okta

Checkmk may work with other SAML integrations as well, but no support can be given for these.

9. ntopng

The ntopng integration supports the following versions of ntopng Professional and Enterprise:

  • >= 5.6

10. Nagvis

Nagvis depends on PHP and requires a minimum version to run. To run Nagvis, please make sure to have a matching version of PHP in your installation:

  • >= 7.2

11. InfluxDB

Checkmk supports the export of data to the following versions of the InfluxDB API:

  • >= 2.0
